## Purpose

V2-C is the dashboard-local Demo Day presentation, choreography, legibility, and reliability layer over already-shipped Meridian proof. It makes the existing snapshot, authority, Foreman/Auth, disclosure, HOLD, absence, and audit evidence easier to present under judge/reviewer conditions.

## No-New-Substance Rule

- V2-C is presentation/choreography/reliability only.
- V2-C reads existing dashboard-local state and committed scenario proof.
- V2-C does not register root/shared skins.
- V2-C does not widen root governance, forensic, authority, live absence, root skins, API, package, auth, deploy, env, config, or secret surfaces.
- V2-C does not claim manual proof that remains HOLD.
