# Foreman audio cues

DEMO-7 ships six dashboard-local static WAV cues for the Foreman presentation layer:

- `idle.wav`
- `explaining.wav`
- `holding.wav`
- `warning.wav`
- `blocked.wav`
- `public-boundary.wav`

These files are locally generated synthetic tones, not voice recordings, voice cloning, runtime TTS, model output, Whisper input, or external service output. They were generated in-repo with Node.js using deterministic sine/silence envelopes and written as 16-bit PCM WAV files at 24 kHz.

Generation method: a local Node scriptlet created each cue from short sine-tone segments with linear attack/release envelopes and no network input. No third-party samples, libraries, external URLs, or licensed source recordings were used.
