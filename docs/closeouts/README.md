# Closeouts Index

## Purpose

This lane stores durable wave/session closeout records.

## What Belongs Here

- Shipped scope summaries
- Explicit non-shipped scope notes
- Remaining HOLDs carried forward
- Signoff status for each wave/session

## Available Closeouts

- [`WAVE1_CLOSEOUT.md`](WAVE1_CLOSEOUT.md)

Future waves should add new closeout files in this directory and list them here.
